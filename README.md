# DevPod Infra Base

This repository defines a DevPod devcontainer image for infrastructure and DevOps work.

Tools installed (via Dockerfile) include:
-  Core system & networking tools
-  Git
-  Python, pip, Go
-  Neovim, ripgrep, fd-find, fzf, tmux
-  Ansible
-  AWS CLI v2
-  Terraform
-  kubectl
-  Helm
-  1Password CLI (op)

Use with:

    devpod up .
